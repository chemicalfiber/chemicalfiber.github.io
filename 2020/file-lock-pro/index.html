File lock Pro-一款文件和文件夹加密工具

发表于2020-05-02更新于2021-03-10

每个人的电脑里总有那么点不相让别人知道的「小秘密」，但是现在的Windows系统并不像手机上的定制UI那样，拥有「程序锁」之类的应用程序。所以，如何在PC上加密这些文件/文件夹就成了一个问题。

在经过化学纤维测试了多款软件之后，决定向大家推荐这款软件：

GiliSoft File Lock Pro

这是个啥？

软件官网上的介绍是这样的：

Why do you need Gilisoft File Lock Pro? No matter if you are a businessman, an employee, a student, or even a housewife, a computer can assist you in your routine work. You save your personal information, documents, and other similar sensitive stuff on your computer that can hurt you if they get compromised. With it you will never worry about data theft by malicious behavior and privacy leaks.
It is a Military Grade Encryption Tool for your files. Lock folders on internal hard drive, flash drive, external USB drive, thumb drive, memory card, pen drive, and network drive. Encrypt files, folders; Hide files folders and drives; Make files folders and drives read only; Password protect files folders and drives. Gilisoft File Lock Pro is available in Chinese, Czech, English, French, Italian, Japanese, Portuguese.

中文翻译（机翻+化学纤维润色）

为什么你需要Gilisoft File Lock Pro？无论你是商人、雇员、学生，甚至是家庭主妇，电脑都能帮助你完成日常工作。你把你的个人信息、文件和其他类似的敏感信息保存在你的电脑上，如果它们被泄露，你将会蒙受损失。有了它，您将永远不必担心恶意行为和隐私泄漏所导致的数据盗用。
它是一个军用级的文件加密工具。锁定内部硬盘驱动器、闪存驱动器、外部USB驱动器、拇指驱动器（U盘）、存储卡、笔驱动器（pen drive,其实还是U盘）和网络驱动器上的文件夹。加密文件、文件夹；隐藏文件、文件夹和驱动器；使文件、文件夹和驱动器只读；对文件、文件夹和驱动器进行密码保护。Gilisoft File Lock Pro提供中文、捷克语、英语、法语、意大利语、日语和葡萄牙语版本。

太长不看版本

这是个文件/文件夹/磁盘加密工具，能加密指定的文件/文件夹/磁盘。

如何使用？

下载文件，关闭安全软件后解压，执行GiliSoft File Lock v11.5.exe进行软件的安装。

安装完毕后会提示重新启动电脑，选择“重启”。

重启之后不要打开软件！打开C:\windows\system32\drivers\etc目录下的hosts文件，在文件底部添加条目
127.0.0.1 gilisoft.com
在此期间，安全软件应一直处于关闭状态。

执行keygen.exe，选择软件GiliSoft File Lock Pro，点击generate，复制生成的激活码，随便输入个邮箱地址，然后粘贴注册码到对应的输入框，点击“激活”即可。




激活之后，便会提示你设置密码，密码不得少于6个字符，越复杂越好。
激活之后，点击“文件加锁”，选择你要加密的文件/文件夹即可。

当你访问已经加密的文件夹/文件时，会有弹窗要求你输入密码以进行访问。当你关闭已经加密的文件/文件夹时，会有弹窗提示是否重新上锁。




点击“设置”>“修改密码”可以修改你的访问密码。

因为密钥是「未经验证的」，所以下一次打开软件时会弹窗进行提示，并将软件重置到「试用版」，不过不用担心，点击右上角的“激活”，重新进行一次破解即可。所以这里化学纤维建议各位一次性把所有要加密的文件夹添加到列表里，就不需要频繁地打开这个软件，频繁地进行激活步骤了。


注意事项

当出现了试用版弹窗的时候，请务必及时激活程序！否则所有密码都会重置为123456！
