我再也不手贱乱来重装系统了

发表于2020-07-19更新于2020-08-22

我就重装个Windows，鬼知道那分区工具会把我的Mac分区格式化啊！！！！

反正现在就非常后悔，非常后悔

事情是这样的，因为我那破笔记本上运行的Windows Server 2019不支持独立显卡，所以打算换成LTSC，然后考虑到我不经常用Windows，顶多就是抢个银行（PAYDAY 2），打打丧尸（Left 4 Dead 2）或者看看云（GTA V），所以就打算将C盘的容量缩一缩，让出一点给D盘装点录制好的视频方便我后期剪辑，不用每次都插外接硬盘（Mac对NTFS的外接硬盘兼容性真差）。
结果我提交分区表的改动之后，这次的操作比上次花费的时间长了许多，我当时并没有特别在意，结果今天早上我想启动MacOS的时候，发现Clover上的Mac启动项只剩下个Windows了。然后我回磁盘分区工具一看，硬盘后边那一截变成了「未分配」而不是「Microsoft Basic Data Partition」。
我当时心想：完了！一定是昨天那超长时间的操作直接给我把Mac分区干掉了，现在那一块就是「空空如也」的状态。

我的项目和环境啊！！！！！！！！！！！！！！！！！
